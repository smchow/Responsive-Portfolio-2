Ultimate Road Trip Planner

The Ultimate Road Trip Planner is an app that helps users in planning a road trip. The user provides a starting location and ending location. The interactive app then makes suggestions for points of interest like Hotels/Motels and restaurants. It also provides weather forecasts for key cities along the route.
